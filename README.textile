h3. Modelo de tese/dissertação IFUSP

Este é um modelo de tese/dissertação para trabalhos de mestrado e doutorado apresentados no Instituto de Física da Universidade de São Paulo (IFUSP). Ele é baseado no bundle Koma-Script.

h3. Arquivos

* .pdf - Documento final de exemplo
* .tcp - Arquivo de projeto (comece por aqui se você usa o TeXnicCenter)
* .tex - O arquivo principal (comece por aqui se você NÃO usa o TeXnicCenter)
* .tps - Arquivo interno do TeXnicCenter (ignore-o)
